# StockAI 智能分析系統 v1.5

一個專業嘅 AI 股票分析系統，整合富途牛牛 API，提供即時股價、技術指標、AI 模型預測同最終交易決策。

---

## 📋 系統功能

| 功能 | 說明 |
|------|------|
| 📊 **即時股價** | 富途牛牛 API 即時報價 |
| 📈 **K線圖 + 保力加通道** | 專業圖表分析 |
| 🎯 **8 個技術指標** | 趨勢、RSI、MACD、成交量、保力加、ATR/ADX、KDJ、OBV |
| 🤖 **多模型 AI 分析** | Claude + DeepSeek 雙引擎對比分析 |
| 💬 **智能問答** | 支持複雜問題回答 |
| 📋 **自動策略生成** | 根據指標自動推薦完整交易策略 |
| 🏆 **最終決策系統** | 融合多信號，自動投票 |
| 📊 **PE 估值刻度尺** | 歷史百分位顯示 |
| 🎨 **靚仔 UI** | 深色主題 + 玻璃態效果 |

---

## 🆕 v1.5 新功能

### 1️⃣ Claude AI 集成
- 支持 Claude Sonnet 4 分析
- 更強嘅推理同理解能力
- 需要配置 API Key

### 2️⃣ 多模型對比分析
- 同時使用 Claude + DeepSeek 分析
- 多数投票機制產生最終信號
- 詳細對比報告

### 3️⃣ 智能問答增強
- 支持複雜問題："多模型對比"、"自動策略"、"綜合評估"
- 根據問題自動選擇合適嘅分析模式

### 4️⃣ 自動交易策略生成
- 綜合評分 (0-100)
- 買入/賣出價位建議
- 止損價設定
- 目標價設定
- 風險回報比計算

## 📦 安裝步驟

### 1. 安裝 Python

- 下載 Python 3.8+：[https://www.python.org/downloads/](https://www.python.org/downloads/)
- 安裝時 **必須勾選** 「Add Python to PATH」

### 2. 下載 StockAI 系統

將整個 `stockai_system_v1.4` 文件夾下載到電腦

### 3. 安裝富途 OpenD（重要！）

**呢個係獲取即時股價嘅關鍵！**

1. 去富途開放平台下載 OpenD：  
   👉 [https://www.futunn.com/download/OpenAPI](https://www.futunn.com/download/OpenAPI)

2. 下載 Windows 版本並安裝

3. **啟動 FutuOpenD**（第一次需要登入富途賬號）

4. 確認 OpenD 右下角托盤圖標顯示「已連接」

> ⚠️ **注意**：OpenD 必須保持運行，StockAI 先可以攞到數據！


雙擊 `start_system.bat` 运行系统
